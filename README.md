# URL to QR converter
Visit the application [here](https://url-to-qr-generator.onrender.com)

## Description
A simple web application that takes in any URL and generates the appropriate QR code, which you can also download. This web application consists of a static frontend and a Node.js backend. While the frontend is deployed via GitHub Pages, the server-side logic is hosted on Render.com.

## Tools
- HTML, CSS (responsive), JS
- Node.js + Express.js (Middleware: body-parser, qrcode)
